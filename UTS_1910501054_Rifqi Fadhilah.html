Desain Web Tutorial : Contoh Desain Web Dengan HTML+CSS

CSS adalah singkatan dari Cascading Style Sheets. Kalau baca di kamus, cascading itu artinya air terjun. Tapi dalam hal ini, yang di maksud adalah, aliran dari suatu kode ke kode lain yang saling berhubungan. Jadi kamu di tulis lengkap dalam bahasa Indonesia kira-kira arti CSS adalah: kumpulan kode-kode yang berurutan dan saling berhubungan untuk mengatur format/tampilan suatu halaman HTML.
Keuntungan Pengunaan CSS
Jika anda memiliki beberapa halaman website dimana Anda menggunakan font arial untuk tulisannya, lalu suaut hari anda bosan dengan arial dan ingin mengganti ke trebuchet, anda harus merubah satu per satu halaman website anda dan merubah tipe font dari ariak menjadi terbuchet.

Desain Web Tutorial : Hypertext Markup Language

HTML adalah sebuah bahasa markup berupa kode-kode tag yang menginstruksi browser untuk menghasilkan tampilan sesuai dengan yang diinginkan.
1. HTMLadalah sebuah standar yang digunakan secara luas untuk menampilkan halaman standar Internet yang saat ini dikendalikan oleh World Wide Web Consortium (W3C).
2. Sebuah file HTML dapat dibuka dengan menggunakan browser web seperti Mozilla Firefox atau Microsoft Internet Explorer.
3. Versi terakhir adalah HTML 4.01. Spesifikasi HTML standar :
